اهداف پروژه:
جلوی حملات XSS گرفته شود. با استفاده از امکانات jinja2 و lxml.
قابلیت ایجاد حساب کاربری، تغییر رمز و authentication.
یک جدول user داریم که و یک جدول post و یک جدول comment، که هر کاربری فقط پست های خودش را می تواند ویرایش یا حذف کند و همچنین می تواند پست ها را برای همه یا فقط برای خودش و یک کاربر دیگر قابل نمایش کند. در این پروژه برخی پست ها برای کاربر teacher قابل نمایش است. و برخی پست ها برای همه.

این پروژه با flask و در pycharm توسعه پیدا کرده. به کمک خودآموز flaskr. برای کار با پروژه همان مراحل متداول که برای flaskr طی می شود کافی است. یعنی نصب نیازمندی های پروژه با pip install -e . و تنظیم FLASK_APP و FLASK_ENV و سپس راه اندازی پایگاه داده با دستور flask init-db و سپس اجرا به کمک flask run.
